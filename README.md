# Inventory System API

## Overview

Inventory management API built with .NET 8 using Clean Architecture, CQRS, MediatR, JWT Authentication and Entity Framework Core.

---

## Architecture

- Clean Architecture
- CQRS Pattern
- Repository Pattern
- JWT Authentication
- FluentValidation
- MediatR
- AutoMapper

---

## Technologies

- ASP.NET Core 8
- Entity Framework Core
- SQL Server
- MediatR
- FluentValidation
- AutoMapper
- Serilog

---

## Project Structure

src/

  ├── Inventory.API
    
  ├── Inventory.Application
    
  ├── Inventory.Domain
    
  └── Inventory.Infrastructure

---

## Features

- JWT Authentication
- Product Management
- Categories
- Purchase Orders
- Sales Orders
- Inventory Tracking
